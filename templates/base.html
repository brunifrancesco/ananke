How Ananke works
Ananke is another single-team poker planning tool.

A poker planning is a method aimed to estimate the effort required to complete a job.

It involves the whole team who expresses the vote and it is widely used for timeboxing Agile principles. More on this here.

This tools works as following
  • Admin/Scrum masters point to root page
  • Voters/team members point to the voting link page
Each voter will have an identifer assigned which follows the following schema:
  • an adjective
  • a crypto name

Once all votes have been received, the master can reveal them to make them visible or reset them to start over.
